# Data provenance

`pokemon.js` holds the full Pokémon Champions roster as shown on the
[Smogon Champions Strategy Pokedex](https://www.smogon.com/dex/champions/pokemon/),
extracted **July 2026** — 323 Pokémon.

Per Pokémon: name, types, abilities, base stats, Smogon tier (`Uber`, `OU`,
`UUBL`, `UU`, `NFE`), and national dex number.

Notable roster facts (as extracted):
- Includes ~120 alternate forms: Champions-exclusive **Mega Evolutions**
  (Mega Garchomp, Mega Meganium, Mega Feraligatr, Mega Raichu X/Y, …),
  regional forms (Alola/Galar/Hisui/Paldea), and cosmetic/battle forms.
- Tier distribution: 8 Uber, 74 OU, 4 UUBL, 236 UU, 1 NFE.

## Refreshing

```bash
curl -s "https://www.smogon.com/dex/champions/pokemon/" -o /tmp/champions.html
python3 scripts/extract_data.py
```

The extractor parses the `dexSettings` JSON blob embedded in the page (the
site is a JS app; the data ships inline). If Smogon changes their page
structure the script fails loudly rather than writing partial data.

## Moveset usage data (`movesets.js`)

Generated by `scripts/extract_movesets.py` from Smogon monthly usage stats
(chaos JSON). Primary source: the Champions VGC doubles ladder
(`gen9championsvgc2026regmb`, 1760+ Elo cut); fallback for mons unranked
there: the 4v4 doubles UU ladder. June 2026 stats cover 282 of 323 mons.

Per Pokémon: top-10 moves with usage % (of teams carrying the mon), most
common item / ability / nature+spread, and overall usage %. Champions
spreads use 0–32 stat units; the extractor converts to 0–252 EVs (×8,
cap 252). `MOVE_META` holds type/category/power for every damaging move.

```bash
python3 scripts/extract_movesets.py 2026-06   # or any published month
```

## Learnsets (`learnsets.js`)

Generated by `scripts/extract_learnsets.py` via Smogon's dex RPC
(`dump-pokemon`) — one request per base form (241 forms, rate-limited).
Megas, battle forms (Aegislash-Blade, Palafin-Hero...), and cosmetic forms
map to their base form's learnset via `LEARNSET_ALIASES`. Used to restrict
the set editor's move list and validate imports.
